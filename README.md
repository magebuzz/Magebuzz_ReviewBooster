# Review Booster
> Get more trustworthy reviews and increase your sales


* Easy to collect more product reviews
* Share and promote reviews on social networks
* Display review on product page and sidebar


## Features

   Review on Product page

* Display reviews in product page
* Allow customers to write review directly on product page
* Allow to turn on / off captcha for review form



   Sidebar Display

* Allow to display reviews in sidebar
* Allow admin to select number of reviews for displaying
* Allow admin to filter reviews by rating and category


   Review Reminder

* Automatically send email to customers for asking review after purchase. 
* Allow admin to configure order status, time and email template.
* Allow admin to manage review reminders
* Customers can submit review directly in the email
* Allow admin to send reminder email manually
* Send only one reminder email if the order has more than one item
  * Reminder status: sent, pending or customer replied.
  * Delete old reminders



   Review promotion

* Notify admin when a new review is posted
* Customers can easily promote their favorite review via Facebook and Twitter
* Customers can vote a review as helpful or not
* Allow admin to post review into Facebook Fan page
